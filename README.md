# JavaTraining-Week2

「プログラムの目的」  
このプログラムの目的は書籍の情報を登録、一覧表示、削除できる簡易的な図書管理アプリです。  
Bookクラスで一冊の本の情報を保持し、Libraryクラスで書籍の管理や削除の処理を担当します。
LibraryAppでユーザーの入力を受け取り、各処理を実行します。

「実行方法」  
1 ターミナル（またはコマンドプロンプト）で、プログラムのあるフォルダに移動します。  
2 次のコマンドでコンパイルします。  
   `javac LibraryApp.java`  
3 コンパイルが成功したら、以下のコマンドで実行します。  
   `java LibraryApp`  
  
（アプリの使用方法）  
実行するとメニューが表示されます。  
   1: 書籍を登録する  
   2: 書籍一覧を表示する  
   3: 書籍を削除する  
   4: 終了する  
1 → 本のタイトル・著者・出版年を入力して登録できます。  
2 → 登録されている書籍一覧が番号付きで表示されます。  
3 → 表示された番号を入力することで書籍を削除できます。  
4 → アプリを終了します。  
  
「クラス設計のポイント」  
・Bookクラス  
書籍一冊分の情報（タイトル名、著者名、出版年）を保持するためのクラスです。  
別のクラスから直接触れないようprivateにしています。

・Libraryクラス  
書籍を登録、一覧表示、削除する機能を担当するクラスです。  
Bookクラスで保持した情報をArrayListで管理し、操作に応じて処理を行います。

・LibraryAppクラス  
ユーザーからの入力を受け付け、Libraryクラスの機能を呼び出して処理を行うクラスです。  
ユーザーからの情報を直接受け取ることができる「メインクラス」の役割を担っています。
