# 内定者研修 課題提出手順書（Fork → PR 提出）

## はじめに
この文書は、研修課題を GitHub を利用して提出するための手順をまとめたものです。  
以下の手順に従って、課題を Fork → 実装 → PR（Pull Request）として提出してください。

---

# 1. 課題リポジトリへアクセス
研修担当者より指定された課題用 GitHub リポジトリへアクセスしてください。

例）https://github.com/ORG/training-week1

---

# 2. リポジトリを Fork する
画面右上の **「Fork」** ボタンをクリックし、自分のアカウントにコピーします。

これにより、あなた専用の課題リポジトリが作成されます。

---

# 3. 自分の Fork したリポジトリを clone（ローカルへ取得）

```bash
git clone https://github.com/<あなたのGitHub名>/training-week1.git
cd training-week1
